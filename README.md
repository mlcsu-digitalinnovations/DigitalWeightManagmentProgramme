# NHS Digital Weight Management — Interoperability Specifications

This repository holds specification documents, examples and conformance artefacts for interoperable integrations with the NHS Digital Weight Management Programme.

## Purpose

Provide a concise, standards-based reference that helps system suppliers, commissioners and implementers exchange referral, enrolment and outcome data between NHS systems (primary care, referral portals, and digital provider platforms).

## Scope

- Referral submission from an authorised healthcare system to a digital weight-management provider
- Outcome data exchange (weight, BMI)
- Conformance, testing and example payloads
